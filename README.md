# Marketing Analytics & Sentiment Analysis Project
The objective of this project is to transform raw marketing data into meaningful insights that support data-driven decision making.

---
## Project Overview:
This project presents an end-to-end data analytics workflow that starts with data cleaning and transformation using SQL, continues with sentiment analysis using Python, and concludes with an interactive Power BI dashboard to generate actionable business insights.

---
## Tools & Skills:
+ SQL → Data Cleaning & Transformation

+ Python (Pandas, NLP Libraries) → Data Analysis & Sentiment Analysis

+ Power BI → Interactive Data Visualization & Dashboard Development

+ GitHub → Project Documentation & Version Control

--- 

## Key Insights
- Conversion rates fluctuated throughout the year, with strong seasonal peaks

- Customer engagement declined during the second half of the year

- Average customer rating: 3.7 (below the 4.0 target benchmark)

- Positive sentiment dominates overall feedback, with opportunities to improve mixed and negative experiences
